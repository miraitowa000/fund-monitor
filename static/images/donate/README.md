# 打赏二维码图片说明

请将你的收款码图片放在此目录下：

## 文件命名要求

1. **微信赞赏码**：`wechat_qr.png`
   - 建议尺寸：300x300 像素
   - 格式：PNG 或 JPG
   - 说明：微信官方赞赏码，扫码后可选择预设金额或自定义金额

2. **支付宝收款码**：`alipay_qr.jpg`
   - 建议尺寸：300x300 像素
   - 格式：PNG 或 JPG
   - 说明：支付宝个人收款码，用户需手动输入金额

## 如何获取微信赞赏码

1. 微信搜索"赞赏账户"小程序
2. 完成实名认证
3. 保存赞赏码图片
4. 重命名为 `wechat_qr.png` 并放到此目录

## 如何获取支付宝收款码

1. 打开支付宝 APP
2. 点击"收钱" → "保存图片"
3. 重命名为 `alipay_qr.jpg` 并放到此目录

## 注意事项

- 确保图片清晰，扫码能正常识别
- 不要在收款码中包含真实姓名等敏感信息
- 如果图片不存在，页面会显示占位符提示
